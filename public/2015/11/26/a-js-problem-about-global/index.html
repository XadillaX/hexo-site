小结

　　最后再梳理一遍。

　　在于 Node.js 的 vm 里面，顶级作用域下的 var 会把变量贴到 global 下面。而 REPL 使用了 vm。然后 $ node 进入的一个模式就是一个特定参数下面启动的一个 REPL。

　　所以我们一开始提出的问题里面在 $ node foo.js 模式下执行是 undefined，因为不在全局变量上，但是启用 $ node 这种 REPL 模式的时候得到的结果是 2。

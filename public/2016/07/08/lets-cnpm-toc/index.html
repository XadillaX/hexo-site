本系列文章将会基于 cnpm 部署搭建一套属于自己（或者公司）的内部私有 NPM 系统。

什么是 CNPM

cnpm 是 Node.js 中国社区成员主导的一个私有 NPM 开源项目，可以用于部署私有 NPM、公共 NPM 镜像等。

阅读对象

  • 有一定 Node.js 后端基础
  • 需要有一套私有 NPM
  • 想要研究 CNPM 实现

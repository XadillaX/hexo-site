## 什么是 CNPM

[cnpm](https://github.com/cnpm/cnpmjs.org) 是 Node.js 中国社区成员主导的一个私有 NPM 开源项目，可以用于部署私有 NPM、公共 NPM 镜像等。

## 阅读对象

+ 有一定 Node.js 后端基础
+ 需要有一套私有 NPM
+ 想要研究 CNPM 实现

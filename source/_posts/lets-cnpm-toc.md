## 什麼是 CNPM

[cnpm](https://github.com/cnpm/cnpmjs.org) 是 Node.js 中國社區成員主導的一個私有 NPM 開源項目，可以用於部署私有 NPM、公共 NPM 鏡像等。

## 閱讀對象

+ 有一定 Node.js 後端基礎
+ 需要有一套私有 NPM
+ 想要研究 CNPM 實現
